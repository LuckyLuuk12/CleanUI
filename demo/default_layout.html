Default Styles

The CleanUI package is mainly a default styling set for my personal projects but in general offers me the following benefits:

  • Consistent design across projects
  • Easy to maintain and update
  • Responsive design for various devices
  • Automatic page layout given a proper HTML structure
The default layout is designed to be simple and clean, providing a solid foundation for building web applications. I hope it ensures I don't make that many more "sketchy" designs in the future. (e.g. using box shadows and gradients everywhere...)

Default styles for proper HTML?

A nice feature of the CleanUI package is that it provides default styles for common HTML elements. So assume you have something like this in your HTML, no real usage of any of my classes at all, just a simple HTML structure: 
<body>
  <header>
    <h1>My Example</h1>
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <h2>Welcome to my example page</h2>
    <p>This is a simple example of how the CleanUI package can style your HTML elements.</p>
  </main>
</body>
Then if you successfully include the CleanUI package, it will automatically apply styles to these elements. And by doing so, it produces a page similar to this one you're looking at without the actual text in the main content. To also get full highlighting support in pre-code blocks you have to include this in your HTML's head: 
<link rel="stylesheet" href="/css/index.css">
<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/highlight.min.js"></script>
<script> hljs.highlightAll(); </script>
